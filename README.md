# Виктор Меламед · Колонка об иллюстрации

Архив-сайт легендарной авторской колонки **Виктора Меламеда** об иллюстрации, которую он вёл на сайте журнала **«[кАк)»** (`kak.ru/columns/illumination`) с октября 2006 по июль 2009.

Оригинальный сайт закрыт. Все **69 выпусков** спасены из [Wayback Machine](https://web.archive.org), очищены и собраны здесь в виде статического журнального сайта.

## Что внутри

- **`index.html`** — оглавление всех 69 выпусков по годам.
- **`a/NN.html`** — каждый выпуск отдельной страницей (полный текст + ссылка на оригинал в архиве).
- **`ideas.html`** — разбор ключевых идей колонки (8 сквозных тезисов с цитатами).
- **`illustrators.html`** — ~130 художников и иллюстраторов, упомянутых Меламедом.
- **`_src/`** — исходники: тексты выпусков в Markdown, метаданные, скрипты сборки (`export.py` — выгрузка из Wayback, `build.py` — генерация сайта).

## Как смотреть

Открыть `index.html` в браузере. Либо, если включён GitHub Pages, — по адресу сайта репозитория.

## Дизайн

Швейцарский журнальный стиль: Helvetica, крупные акциденции, плотный набор, красный акцент — оммаж вёрстке журнала «Как)».
